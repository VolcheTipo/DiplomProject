# 🛡️ Network Anomaly Detector

Гибридная система обнаружения аномалий в сетевом трафике на основе нескольких ML-методов и нечёткой логики.

## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Генерация примеров данных
```bash
python data/generate_samples.py
```

### 3. Обучение ML-модели
```bash
python -m app.train_ml_model
```

### 4. Запуск веб-интерфейса
```bash
streamlit run ui/streamlit_app.py
```

### 5. Опционально запуск API (НЕДОРАБОТАНО)
```bash
python run.py 
# API доступно на http://127.0.0.1:8000
# Документация: http://127.0.0.1:8000/docs
```

## Детекторы

| Метод | Принцип | Особенность                             |
|---|---|-----------------------------------------|
| **KMeans** | Расстояние до центроида кластера | Быстрый, хорош для компактных кластеров |
| **Isolation Forest** | Изоляция аномалий деревьями | Эффективен на высоких размерностях      |
| **One-Class SVM** | Граница нормального поведения | Хорош при малом числе аномалий          |
| **Statistical** | Z-score отклонение | Простой, интерпретируемый               |
| **Random Forest** | Предобученный классификатор | Учитывает помеченные данные             |
| **Hybrid** | Нечёткая агрегация всех методов | Финальная оценка                        |

## Форматы входных данных

**CSV** :
```
src_ip, dst_ip, src_port, dst_port, protocol, bytes, packets
192.168.1.10, 8.8.8.8, 54321, 443, TCP, 15420, 22
```

**PCAP**: стандартный захват трафика (Wireshark, tcpdump).

## Типы обнаруживаемых атак

- **Port Scan** — сканирование портов (мало байт, много соединений на разные порты)
- **DDoS** — огромный объём трафика с одного источника
- **Data Exfiltration** — большой исходящий трафик на нестандартные порты
- **Brute Force** — множество попыток подключения к SSH/RDP
